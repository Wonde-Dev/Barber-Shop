# ✂️ Blade & Co. - Premium Barber Shop Website

A modern, fully responsive barber shop website featuring a video background, animated testimonials, service cards, contact form, and a dynamic theme switcher. Built with pure HTML, CSS, and JavaScript.

## 📋 Features

- ✅ **Full Responsive Design** - Works perfectly on desktop, tablet, and mobile devices
- 🎬 **Video Background** - Engaging hero section with autoplay video overlay
- 📱 **Mobile Menu** - Animated sliding menu with smooth transitions
- ⭐ **Testimonial Carousel** - Auto-sliding customer reviews with manual controls and dot indicators
- 🎨 **Dynamic Theme Switcher** - Toggle between dark and light themes with one click
- 💈 **Service Cards** - Animated cards displaying all barber services with pricing
- 📍 **Google Maps Integration** - Shows Addis Ababa, Ethiopia location
- 📧 **Contact Form** - Interactive form with validation and feedback messages
- 🖼️ **Animated Image Gallery** - Auto-scrolling gallery in the about section
- 🔄 **Smooth Scroll Navigation** - Seamless page navigation between sections
- ✨ **Scroll Reveal Animations** - Elements fade in as you scroll down

## 🛠️ Technologies Used

| Technology | Purpose |
